Website: movie favorites.

A simple movie trailer website to exercise some programming foundations with python.

Note: This is a solution to project 1 of the Udacity Full Stack Web Developer Nanodegree based on the course Programming Foundations with Python (ud036). The solution is graded with "Exceeds Specifications".

Detailed Description

The website shows nine of my favorite movies with additional infos pulled from the OMDb API:

  • Poster image
  • Title
  • Year
  • List of genres separated by dashes
  • Plot with maximum of 150 characters (if longer, truncated with '...').
  • imdb-Rating

Additionally to the OMDb-Infos, an English trailer from YouTube is added for each of the movies.

Requirements

Python 2.7 installed

Running Instructions

Generate the HTML page "fresh_tomatoes.html" by running

python entertainment_center.py

from the command prompt. By clicking on the movies, a modal opens up and the trailer is played.
